# APH
O sistema é referente a um ambiente de preenchimento e controle de preferências de horários de professores. Assim, a partir da usa integração com o Sistema Unificado de Administração Pública (SUAP) 
, Assistente de Carga Horária Acadêmico (ACHA) e TimesTables, ele ajudará no processo de distribuição de horários aos docentes.

> APH é um acrônimo para Assistente de Planejamento de Horários

Atualmente, as funcionalidades que cabem a este sistema são: <br/>
* Cadastro de preferências de horários respeitando regras pré-estabelecidas
* Análise e edição das preferências enviadas pelos professores
* Definição de horário limite para envio das preferências 
* Controle dos usuários do sistema ao nomeá-los como membros da comissão
