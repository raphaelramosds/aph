# Administrador de horários acadêmicos do IFRN, Campus Natal Zona Norte

Trata-se de um sistema conectado a API do Sistema Unificado de Administração Pública, onde os docentes do IFRN 
poderão cadastrar as suas **preferências de horários**. Faz-se necessário seu desenvolvimento porque 
há necessidade de dinamizar o atual processo de registro de preferências, tanto na parte dos professores como no lado da comissão de horários. 

> APH é um acrônimo para Assistente de Planejamento de Horários
